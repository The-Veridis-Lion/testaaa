# 🧩 插件简介

这是一个用于**实时净化与深度清理文本内容**的关键词过滤插件。当前版本已支持多种匹配模式、规则合集管理、全域数据洗刷、移动端适配，以及更完整的安全防护与兼容处理，适用于日常聊天净化、规则替换和存档清理等场景。

# 🚀 大更新

## 🚀 v5.0.0 异步清理与跨存档转移

### ⏱️ 深度清理与性能进化
* **异步防卡死机制**：深度清理改为分片执行，大幅降低浏览器假死风险。
* **可视化进度展示**：新增进度面板，清理阶段与命中状态一目了然。
* **智能超时保护**：支持超时自动中止，避免长时间卡住无法操作。

### 🔒 兼容性与安全增强
* **全局防递归保护**：净化过程加入状态锁，避免互相触发导致循环崩溃。
* **数据库级兼容**：现已支持与数据库外置类插件协同使用。
* **旧版格式自动兼容**：导入时可自动识别并转换早期遗留规则格式。

### 🔀 规则管理升级
* **规则跨存档转移**：支持将规则合集一键复制或移动至其他预设。
* **更智能的词条解析**：目标词与替换词分开处理，导入与编辑更准确。
* **移动端排版修复**：优化长文本显示，避免超长内容撑破界面。

## 🧩 v4.0.0 三擎驱动与全域净化

### ⚙️ 核心能力升级
* **三大净化引擎**：支持普通文本、专业正则与简易组合三种模式，可覆盖从基础替换到复杂规则匹配的多类需求。

### 🗂️ 规则与预设系统重构
* **多组规则管理**：支持按合集分组管理，并可独立开关。
* **完整预设体系**：支持新建、重命名、本地保存，以及 JSON 导入导出。
* **可视化编辑面板**：新增独立编辑界面，可直接预览规则模式与映射关系，长规则也更易查看和维护。

### 🪐 深度清理范围扩展
* **全域数据净化**：深度清理不再局限于聊天记录，现已支持同步洗刷角色卡、世界书与用户自身人设等内容。

---

## ⚙️ v3.0.1 规则系统重构与界面升级

### 🧠 规则逻辑升级
* **多对多规则映射**：从单一屏蔽词列表升级为规则组，可为一组目标词配置一组替换词。
* **随机替换机制**：命中目标词后可随机抽取替换内容；未设置替换词时则默认删除。
* **旧数据自动迁移**：旧版屏蔽词可自动无损转换为新版规则结构。

### 🖥️ 展示与交互优化
* **全新垂直排版**：规则展示改为更直观的上下结构。
* **长内容完整浏览**：支持横向滚动，不再依赖省略号截断超长规则。

### ⚡ 性能优化
* **正则缓存机制**：降低高频输入场景下的重复计算开销，持续使用更流畅。

## 📦 v2.1.0 - v2.0.0 深度清理与批量管理升级

### 🧹 核心清理能力
* **全频段拦截**：可实时监控聊天、思维链与 HTML 注释，命中后即时清理。
* **表格深度兼容**：支持清理表格插件中的单元格、输入框与底层存储内容。
* **深度清理模式**：可同时洗刷聊天记录、插件私有数据与扩展设置，并自动避开自身配置，防止误删。

### ✍️ 输入与批量管理优化
* **无感输入保护**：净化时自动跳过当前正在输入的区域，避免跳光标与打字卡顿。
* **批量导入支持**：支持一次性添加多个屏蔽词，并识别空格、逗号、顿号与换行等多种分隔方式。
* **自动查重过滤**：批量导入时会自动跳过空行与重复词汇。

### 🖼️ 界面与体验提升
* **独立批量输入弹窗**：编辑长列表更方便，同时保留原有单独添加与深度清理功能。
* **移动端适配与单例保护**：优化弹窗表现并修复重复绑定问题，确保逻辑只运行一次。
* **清理后自动刷新**：完成深度清理后自动重载页面，保证前端显示与底层数据同步。

# ✨ 小更新

## 🚀 v5.2.0 角色绑定与流式稳定替换

### 🧷 预设绑定能力
* **默认预设绑定**：新增“默认预设”配置，角色无单独绑定时自动套用。
* **角色卡单独绑定**：可在聊天界面直接为当前角色绑定专属预设，切换角色时自动切换。
* **绑定数据自清理**：重命名、删除预设时会同步维护绑定关系，防止悬空引用。

### 🧩 简易语法优化
* **`*` 限制在短句内**：通配符遇到句号、逗号、分号、换行等停顿符即停止，不再跨句乱删。

### 📺 流式显示逻辑优化
* **视觉替换稳定化**：流式输出中改为稳定替换策略，命中后不会在多次渲染中反复变词。

### 🛡️ 规则安全补丁
* **空匹配自动拦截**：会自动忽略可匹配空字符的危险规则，避免死循环与卡死问题。

### 📲 流式输出优化
* **文本节点强制合并**：针对移动端流式输出中的文本碎裂问题进行处理，提升长句命中率。

### 🎨 界面排版优化
* **大屏显示更舒适**：放大弹窗与字号。长规则名自动换行，模式徽章宽度与对齐方式更稳定。

## 🛡️ v3.2.1 核心防护与安全锁

### 🧱 核心防护区
* **全局 UI 白名单**：将预设菜单、状态栏、抽屉、弹窗与角色编辑界面划入保护范围，避免系统关键区域被误处理。
* **智能输入放行**：在保护区内输入内容时会自动跳过净化，修改预设或设置时不会再被误替换。

### ⚠️ 深度清理安全锁
* **倒计时二次确认**：新增确认弹窗与倒计时提醒，降低误触深度清理的风险。
* **操作引导补充**：清理与同步完成后会提示安全切换流程，减少误操作带来的配置损失。

### 🔧 底层兼容优化
* **滑动分支全面兼容**：重写清理逻辑后，可同时适配新旧结构下的滑动分支内容。
* **智能标点剥离**：录入目标词时可自动处理常见中英文引号，减少因输入格式不统一造成的匹配失效。
